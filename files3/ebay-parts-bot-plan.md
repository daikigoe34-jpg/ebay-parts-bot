# eBay 日本車パーツ輸出リサーチBot 計画書

## プロジェクト概要
日本の自動車純正部品をeBayで海外販売するための「売れ筋リサーチBot」を構築する。

## 自分のレベル
- プログラミング初心者 / 金融・貿易初心者 / MacBook / Claude Code使用

## 手数料の正確な内訳

### ① eBay落札手数料（FVF: Final Value Fee）
出典: https://www.ebay.co.jp/knowledge/fee1/
※商品価格 + バイヤーが払った送料の合計額に対してかかる

| 条件 | FVF率 | 備考 |
|------|--------|------|
| ストア契約なし（大半のカテゴリ） | 13.25%〜15% | カテゴリにより変動 |
| ストア契約あり（Basic以上） | 12.35%〜14.95% | カテゴリにより変動 |
| 自動車パーツカテゴリ（ストアあり） | 12.35% | 他カテゴリより安め |

**+ 固定落札手数料**: $0.30（$10以下の取引）/ $0.40（$10以上の取引）

**重要**: FVFは「商品価格 + バイヤー負担の送料」の合計に対してかかる。
送料無料にして商品価格に上乗せしても、送料別にしても、FVFの対象は同じ。

**初心者向けデフォルト（ストアなし・大半のカテゴリ）: 13.25% + $0.40**

### ② eBay海外決済手数料（International Fee）
出典: https://www.ebay.co.jp/faq2/fee2

- 日本セラーが日本国外に販売する場合に発生
- **基本料率: 1.35%**（売上総額に対して）
- ボリュームディスカウントあり:
  - 月間売上$3,000〜$10,000: 1.20%
  - 月間売上$10,000〜$25,000: 1.10%
  - 月間売上$25,000〜$100,000: 0.95%
  - 月間売上$100,000以上: 0.80%

**初心者向けデフォルト: 1.35%**

### ③ eBay出品手数料（Insertion Fee）
- ストアなし: 月250品まで無料、超過分は$0.35/品
- Basic Store ($21.95/月): 月1,000品まで無料
- **初心者は250品以内で運用 → 実質$0**

### ④ Payoneer為替手数料（USD→JPY引き出し）
出典: https://www.payoneer.com/ja/about/pricing/

- eBay → Payoneerへの入金: **無料**（マーケットプレイス経由）
- Payoneer → 日本の銀行口座への出金（USD→JPY両替）: **最大2%**
  - Morningstarの仲値に対して2%以下の為替手数料を上乗せ
  - 実質的に為替レートが約2円/ドル悪くなる
- $100未満の入金時: $1の追加手数料
- 年間アカウント手数料: $29.95（12ヶ月で受取額$6,000未満の場合のみ）

**デフォルト: 2%**

### ⑤ DDP関税（アメリカ宛・自動車部品）
出典: https://ebay.co.jp/tariffs（2025年8月更新）

- 日米合意により自動車・自動車部品は **一般税率(MFN 2.5%)含め一律15%**
- 通商法第232条の対象 → 第122条関税(10%)とは別枠
- SpeedPAKのDDP見積もりに自動上乗せ
- **デフォルト: 15%**
- ※関税政策は流動的。https://ebay.co.jp/tariffs を定期確認

### ⑥ SpeedPAK Economy送料（アメリカ宛）
- ~100g: ¥1,227〜  / ~500g: ¥1,600〜¥2,000 / ~1.0kg: ¥2,500〜¥3,020
- 容積重量 = 縦×横×高さ(cm) ÷ 8000（実重量と大きい方を適用）
- 対応国: 米国/英国/ドイツ/豪州

### ⑦ その他固定コスト
- 梱包材: ¥300
- 国内送料（郵便局/ローソン持ち込み）: ¥0

## 利益計算ロジック（正確版）

```
【売上】
eBay販売価格(USD) × 為替レート(JPY/USD) = 売上(JPY)
※為替レートはconfig.pyで管理。デフォルト 1USD = 150JPY

【コスト内訳 - eBay側で天引き（USDベース）】
A. FVF落札手数料:    (商品価格USD + 送料USD) × 13.25% + $0.40
B. 海外決済手数料:    (商品価格USD + 送料USD) × 1.35%
   → eBay手数料合計 ≒ 販売総額の約14.6% + $0.40

【コスト内訳 - Payoneer引き出し時】
C. Payoneer為替手数料: 受取額(USD) × 2%
   ※実装上は「為替レートを2%悪くする」で計算
   → 実効為替レート = 仲値レート × 0.98

【コスト内訳 - DDP関税（SpeedPAK経由で支払い）】
D. DDP関税:          商品申告価格(USD) × 15%

【コスト内訳 - 日本側の支出（JPYベース）】
E. 仕入原価:          モノタロウ購入価格（デフォルト ¥2,000）
F. SpeedPAK送料:     重量帯テーブルから算出
G. 梱包材:            ¥300

【利益計算】
eBayからの受取額(USD) = 販売価格 - FVF - 海外決済手数料
日本円での受取額(JPY) = 受取額(USD) × 実効為替レート(仲値×0.98)
利益(JPY) = 日本円受取額 - DDP関税(JPY換算) - 仕入原価 - SpeedPAK送料 - 梱包材
利益率 = 利益 / (販売価格USD × 仲値レート) × 100
```

### 損益分岐の試算（$50の部品、500gの場合）
```
販売価格: $50（送料込み想定）
仲値レート: 150JPY/USD

[eBay天引き]
  FVF: $50 × 13.25% + $0.40 = $7.03
  海外決済: $50 × 1.35% = $0.68
  eBay手数料合計: $7.71
  → eBayからPayoneerへ: $42.29

[Payoneer引き出し]
  実効レート: 150 × 0.98 = 147JPY/USD
  日本円受取: $42.29 × 147 = ¥6,217

[DDP関税]
  $50 × 15% = $7.50 → ¥7.50 × 150 = ¥1,125
  ※SpeedPAK経由で別途請求

[日本側コスト]
  仕入: ¥2,000 / SpeedPAK送料(500g): ¥1,800 / 梱包: ¥300

[利益]
  ¥6,217 - ¥1,125 - ¥2,000 - ¥1,800 - ¥300 = ¥992
  利益率: ¥992 / ¥7,500 = 13.2%
```
